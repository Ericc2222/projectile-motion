Sound Files for Clown Launcher Game

Place the following MP3 files in this directory:

1. cannon_fire.mp3 - Sound effect for when the cannon fires
2. bounce.mp3 - Sound effect for when the clown bounces on the mattress
3. crash.mp3 - Sound effect for when the clown misses/crashes
4. success.mp3 - Celebratory sound for a successful landing
5. level_up.mp3 - Exciting sound for advancing to the next level
6. whoosh.mp3 - Whooshing sound for the clown flying through the air

If no sound files are provided, the game will use placeholder silent sounds.

You can find free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://elements.envato.com/sound-effects
- https://www.zapsplat.com/

Remember to respect licensing terms when using sounds from these sites.
